Powerful features, simple interface

Pro-curo brings together everything you need to manage tissue traceability, samples, and quality workflows — all from a clean, modern platform designed for HTA-regulated establishments.

New in V5

Interactive Dashboard

Get a bird's-eye view of your operations the moment you log in. The dashboard displays live KPIs, project status summaries, and trend charts — all updating in real time.

  • Customisable widgets and layout
  • Real-time data from across your system
  • Quick-access links to recent items
  • Visual charts for trends and workload

Location Tree & Detail Panel

Navigate your entire site hierarchy with an intuitive tree view. Click any location to see its details, linked samples, documents, and history in a side panel — no page reloads, no hunting through menus.

  • Hierarchical site and location structure
  • Inline detail panel for quick access
  • Linked samples, documents, and notes
  • Drag-and-drop organisation

Sample Management

Register, track, and manage samples from intake to disposal. Attach documents, assign to projects, and maintain a complete chain-of-custody record — all with full audit trail.

  • Full sample lifecycle tracking
  • Document and photo attachments
  • Barcode and QR code support
  • Configurable sample types and statuses

Read our white paper: The Importance of Sample Tracking in Life Science Laboratories

Project Management

Organise work by project, client, or site. Assign tasks, set deadlines, track progress, and link everything back to the right samples and locations. No more juggling spreadsheets.

  • Project-level overview and tracking
  • Task assignment and deadlines
  • Link samples, locations, and documents
  • Client and site grouping
New in V5

Reports & Query Builder

Build custom data queries without writing a line of code. The visual query builder lets you filter, group, and sort your data, then export to CSV or generate polished PDF reports.

  • Visual drag-and-drop query builder
  • Save and share custom reports
  • Export to CSV and PDF
  • Schedule recurring reports

Advanced Search

Find any record in seconds. Search across samples, projects, locations, and documents with powerful filters. Save frequent searches for quick re-use.

  • Cross-entity global search
  • Advanced filter combinations
  • Saved search presets
  • Instant results as you type
New in V5

CSV Import Wizard

Migrating data or importing bulk records? The step-by-step import wizard maps your CSV columns to Pro-curo fields, validates your data before import, and gives you a clear summary of what was imported.

  • Drag-and-drop CSV upload
  • Automatic column mapping
  • Pre-import validation and error highlighting
  • Import history and rollback

Built on modern technology

Pro-curo V5 is built from the ground up with a modern, enterprise-grade stack — fast, secure, and ready to scale.

⚙️

React & Ant Design

A fast, responsive front-end built with React and the Ant Design component library for a consistent, professional user experience.

🖥

Node.js Backend

A robust Node.js API with JWT authentication, role-based access control, and comprehensive input validation.

🗃

PostgreSQL Database

Enterprise-grade PostgreSQL for reliable, ACID-compliant data storage with full-text search capabilities.

📦

Docker Deployment

Containerised deployment with Docker for consistent, repeatable installations — whether cloud or on-premise.

📄

PDF Reporting

Generate professional PDF reports directly from your data with customisable templates and automatic formatting.

📋

Full Audit Trail

Every change is logged — who did what, when, and what the previous value was. Complete traceability for compliance.
